# 🎓 월드프렌즈 ICT 봉사단 - 라오스 국립관광대학(NIT) 안드로이드 앱 개발 프로젝트

본 저장소는 2018년 과학기술정보통신부 주관, 한국지능정보사회진흥원(NIA)에서 주최한 **월드프렌즈 ICT 봉사단** 단원으로 라오스에 파견되었을 당시 수행한 안드로이드 모바일 애플리케이션 개발 산출물 및 공식 활동 문서를 정리한 저장소입니다.

파견 당시 모바일 앱 개발(Android) 경험이 전무한 상태에서, 파견 한 달 동안 현지 요구사항 분석부터 독학을 통한 개발, 배포까지 성공적으로 완수한 도전과 성장의 기록입니다.

## 🛠 주요 기술 스택 및 기능

* **Android SDK & Java**: 안드로이드 표준 Java 개발 환경 기반
* **Retrofit 2 & Gson**: 라오스 국립대학 공식 웹사이트(`http://www.nit.gov.la`)의 WordPress REST API(`/wp-json/wp/v2/posts`)를 비동기로 호출하고 JSON 데이터를 객체화하여 처리
* **RecyclerView**: 스크롤 성능이 뛰어난 리스트 UI 구조를 사용해 웹 뉴스를 가독성 있게 표출
* **다국어 대응 및 회원 관리**: 현지 특성을 고려하여 라오어 버전을 분리 탑재하였으며, 회원 가입 관련 비즈니스 로직 연동

## 🏆 주요 성과 및 배운 점

1. **무에서 유를 창조한 실행력**: 안드로이드 지식이 없는 상태에서 한 달 만에 실서비스 수준의 모바일 앱을 구현하며 **빠른 기술 습득 능력(Learning Curve)**과 **끈기**를 증명했습니다.
2. **외국 정부 기관과의 협업 경험**: 라오스 정보통신부 및 관광청 산하 국립기관(NIT) 담당자들과 직접 소통하며 요구사항을 수집하고 솔루션을 조율하는 글로벌 협업 감각을 길렀습니다.
3. **공식 증빙 자료**: 활동 기간 동안 기획했던 IT 프로젝트 세부계획서와 최종 결과보고서 등 공식 문서를 소스코드와 함께 아카이빙하여 역량의 진위성과 프로젝트 완성도를 함께 보증합니다.
